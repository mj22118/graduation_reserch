# 卒業研究
囲碁エージェントKataGoを用いた棋譜からの囲碁棋力の推定と検証

## 概要
* データに含まれるノイズを除去することで，棋力推定の高精度化を目指した．
* 提案手法の有効性を示すため，実際にオンライン対局サイトの棋譜を用いて検証した．

## フォルダ構成
* Analysis: 棋譜の解析
* Estimate: モデル構築およびランクの推定
* Select  : 棋譜の抽出

## 実行環境
* OS: macOS Sonoma 14.8
* 言語: Python 3.9.6
* ライブラリ:
  chardet(5.2.0)，
  pip(25.3)，
  sgf(0.5)
